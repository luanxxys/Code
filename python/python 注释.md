### 关于 #!/usr/bin/env python

它告诉 LINUX/UNIX 去找到 python 的编译器

### 关于 # -*- coding: utf-8 -*-

告诉　python interpret 如何解释字符串的编码，当你的文件中出现中文的时候，你必须使用它

### 文档注释

1． 在必要的地方加上单行注释　＃

1. 每一个函数下面务必加上多行注释　''' code '''

1. 也可以连续多次使用 # 单行注释来代替多行注释，但并不推荐那么做
